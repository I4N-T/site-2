2D Map and Dungeon Generator

Description:

The 2D Map and Dungeon Generator procedurally generates 2D tile world maps and dungeons based on user inputs. The program will generate a map based on the user's choice of any one of four unique algorithms. Each generated map will have a dungeon placed in a random location, which the user can then click on to procedurally generate a dungeon layout.

Tech: C#, Unity

2D Map and Dungeon Generator

Description:

The 2D Map and Dungeon Generator procedurally generates 2D tile world maps and dungeons based on user inputs. The program will generate a map based on the user's choice of any one of four unique algorithms. Each gnerated map will have a dungeon placed in a random location, which the user can then click on to procedurally generate a dungeon layout.

Tech: C#, Unity

Stranded on an Island Simulator

Description:

The Stranded on an Island Simulator is a fun project I created because I wanted to learn how to program desktop applications. The program allows the user to create several survivors by inputting certain information/stats before proceeding to simulate the actions of each of the survivors as they are stranded on an island. This was one of my earliest projects, and it was the first program I created without using Unity.

Tech: C#, Windows Forms
